== Installation ==

= Using The WordPress Dashboard =

1. Navigate to the 'Add New' in the plugins dashboard
2. Search for 'Under Construction'
3. Click 'Install Now'
4. Activate the plugin on the Plugin dashboard

= Uploading in WordPress Dashboard =

1. Navigate to the 'Add New' in the plugins dashboard
2. Navigate to the 'Upload' area
3. Select `under-construction.zip` from your computer
4. Click 'Install Now'
5. Activate the plugin in the Plugin dashboard

= Using FTP =

1. Download `under-construction.zip`
2. Extract the `under-construction` directory to your computer
3. Upload the `under-construction` directory to the `/wp-content/plugins/` directory
4. Activate the plugin in the Plugin's dashboard


== Frequently Asked Questions ==

= Disabled Under Construction Plugin and it still shows =

This is always a caching issue. Go to the caching plugin's setting page and clear the cache.
